# 登革熱疫情擴散App

### 前言

在[單元01](/ex01/README.md)中，我們以ArcGIS Online圖台來呈現2015年於臺南市爆發的登革熱疫情。在這個單元中，我們將延續[單元01](/ex01/step-06.md)的地圖成果，套用時間軸網頁樣板，讓登革熱疫情擴散地圖得以分享。

### 目的

使用ArcGIS Online時間軸網頁樣板，分享登革熱疫情擴散地圖。

### 您將學會

* 套用Time Aware網頁樣板
* 設定網頁樣式

### 需求軟體

ArcGIS Online 訂閱帳號
